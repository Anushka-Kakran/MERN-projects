# 🛒 ShoppyGlobe - Full Stack E-Commerce Application

ShoppyGlobe is a robust, full-stack e-commerce platform built using the **MERN** stack. It features secure user authentication, a dynamic product catalog, and a persistent shopping cart system synced with a MongoDB database.

---

## 🚀 Features

- **User Authentication**: Secure Signup and Login using **JWT** (JSON Web Tokens) and **BcryptJS** for password hashing.
- **Product Management**: Browse a wide range of products with detailed views and real-time search.
- **Persistent Cart**: Add, update, or remove items from a cart that stays saved to your account across sessions.
- **Professional Profile**: Dedicated Account page to view user details like username, email, and joined date.
- **Global Search**: Real-time product filtering using **Redux Toolkit**.
- **Responsive UI**: Fully mobile-friendly design built with **Tailwind CSS** and **React Icons**.

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (Vite)
- Redux Toolkit (State Management)
- Tailwind CSS (Styling)
- React Router DOM (Navigation)
- Axios (API Fetching with Interceptors)

**Backend:**
- Node.js & Express.js
- MongoDB & Mongoose (NoSQL Database)
- JSON Web Tokens (Authentication & Authorization)
- BcryptJS (Password Encryption)
